# ft_trancendence

The final project in 42 common core.

## Docker

To run to project on local environment

```sh
make
```

to run only `Backend NestJS`

```sh
make run-backend
```

to run only database

```sh
make run-database
```

to stop docker compose service

```sh
make down
```

to restart entire service

```sh
make re
```
